Geo Graffiti Api

Find Street art throughout space and time

/user

POST

New users can be created with the following object:

  • firstName (string)
  • lastName (string)
  • email (string)
  • password (string must be 8 characters)

Successful requests will recieve a response with sanitized user information (no password).

/auth/login

POST

User can log in by posting:

  • email
  • password

Successful logins will recieve a json web token valid for one year

/image

POST

Images can be posted as multi-part form data:

  • lat (latitude image was taken at, number)
  • lon (longitude image was taken at, number)
  • time (time image was taken at, number)
  • file (image file)

    • Valid JWT should be set in "Authorizer" header

GET

A get request returns all images on in the database.
